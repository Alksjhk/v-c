# 连接状态组件 (ConnectionStatus)

## 功能说明

ConnectionStatus 组件在前端右上角显示服务器连接状态，提供实时的连接状态监控。

## 特性

- **实时监控**: 每30秒自动检查服务器连接状态
- **视觉反馈**: 使用颜色和图标直观显示连接状态
- **手动刷新**: 点击状态指示器可手动检查连接
- **响应式设计**: 适配不同屏幕尺寸
- **暗色主题**: 支持系统暗色主题
- **智能显示**: 只在用户登录后显示

## 状态说明

- 🟢 **服务器已连接**: 绿色，表示服务器连接正常
- 🔴 **服务器连接失败**: 红色，表示无法连接到服务器，会有脉冲动画
- ⏳ **检查中...**: 黄色，表示正在检查连接状态

## 使用方法

组件已自动集成到 App.tsx 中，无需额外配置。

```tsx
import ConnectionStatus from './components/ConnectionStatus';

// 在 App 组件中使用
<ConnectionStatus />
```

## 技术实现

- 使用 `messageAPI.getLatestMessages()` 测试服务器连接
- 集成 ChatContext 获取用户登录状态
- CSS Modules 实现样式隔离
- 支持点击重新检查连接

## 样式特点

- 毛玻璃效果背景
- 平滑的悬停动画
- 现代化的圆角设计
- 适配移动端显示
